# 网盘搜索   提供网页选中右键选中网盘搜索

## 功能介绍

一键网盘搜索，在百度搜索时一键支持网盘搜索.右键网盘搜索，在任意页面选中需要搜索的关键字，右键网盘搜索

Chrome插件
